데이터베이스 종합 관리 솔루션

데이터베이스 종합 관리 솔루션

오렌지는 데이터베이스 관리 업무를 단순화하고 개발 속도를 높여 생산성을 향상시키는 데이터베이스 개발 및 관리 솔루션입니다.

개발 생산성 향상

데이터베이스 개발에 필요한 다양한 도구와 실효적인 편의 기능을 갖추었습니다. 비즈니스 요구사항 충족을 위한 설계, 개발, 디버깅, 튜닝에 이르는 전반적인 업무 싸이클을 단축시키고 스마트한 업무 환경을 구성하십시오.

단일 프로그램으로 동시에 다양한 DBMS 지원

ODBC 뿐만 아니라 JDBC 접속 지원

Orange Ade 3.0

ODBC + JDBC 접속 지원

  • ODBC 뿐만 아니라 JDBC를 통한 접속을 지원

Schema Browser

  • 데이터베이스에 존재하는 객체 정보를 사용자 별 트리 형태로 일목요연하게 표현

SQL Tool

  • 다수의 SQL 문장을 일괄 또는 순차 실행하거나 특정 SQL 만을 선택적으로 실행할 수 있고 편집 모드 실행을 통하여 실행 결과를 직접 수정 기능 제공
  • 여러 SQL의 다양한 실행 옵션
  • Workspace 기능
  • SQL 의 저장, 불러오기, 실행기능
  • SQL 실행결과 GRID 형태, Text, Excel 형태 파일로 출력 및 정렬기능
  • 미려한 SQL 포맷기능(SQL Formatter)
  • Fetch As Need 기능
  • 자동완성 기능 (Hive 제외)
  • Syntax Highlight 기능
  • 다양하고 편리한 Edit 기능
  • SQL 을 Visual Basic, Power Builder, C/C++, Java Delphi, Perl, PHP 코드로 변환 및 역 변환 기능 제공
  • SQL History 기능. Cached SQL, Named Script 제공

Table Editor

  • 테이블 데이터의 편집(Insert, Update, Delete)과 조건에 의한 일부 데이터 조회 및 저장 기능을 제공합니다. 변경된 Grid 셀은 각각의 변경 타입에 따라 색상으로 구분되어 Commit 또는 Rollback 작업이 직관적이고 편리합니다.

Description Tool

  • 데이터베이스 객체의 상세 정보(객체 생성 스크립트, 기본 속성, Column, Index 정보, Trigger 정보 등)를 제공

Database Information

  • 데이터베이스 기본 설정 정보를 제공하여 데이터베이스의 각종 운영 정보 및 Parameter 운영환경 확인

Query Builder

  • GUI환경에서 마우스 조작만으로 오브젝트의 추가 및 오브젝트들 간의 관계 설정을 통해 SQL 문장을 생성, 편집을 할 수 있는 기능을 제공

Load Tool / Unload Tool

  • 테이블을 선택하거나 쿼리 작성을 통하여 txt, csv, xls, xlsx,json, ldrd 등 다양한 포맷의 파일로 데이터를 내려받거나 외부 파일로부터 테이블에 데이터를 저장할 수 있는 기능을 제공

Windows 7/8/8.1/10/11 Windows Server 2012/2016e

지원환경(DBMS)
