데이터베이스 종합 관리 솔루션

오렌지는 데이터베이스 관리 업무를 단순화하고 개발 속도를 높여 생산성을 향상시키는 데이터베이스 개발 및 관리 솔루션입니다.

관리자를 위한 편의 도구

다양한 모니터를 제공하여 데이터베이스 성능 병목 현상의 진단을 위한 실용적인 접근을 지원합니다. 또한 이 슈 해결을 위한 일련의 도구를 제공합니다. 데이터베이스 변경과 테스트, 데이터 이관과 백업, 자원 관리 등 일상적인 관리 업무도 손쉽게 처리하실 수 있습니다.

개발 생산성 향상

데이터베이스 개발에 필요한 다양한 도구와 실효적인 편의 기능을 갖추었습니다. 비즈니스 요구사항 충족을 위한 설계, 개발, 디버깅, 튜닝에 이르는 전반적인 업무 싸이클을 단축시키고 스마트한 업무 환경을 구성하십시오.

데이터 품질 관리

데이터 분석을 위한 품질 관리 기능을 제공합니다. 프로파일링 중복 및 패턴 검사로 정보의 분석과 이를 통한 데이터 정리 및 변경을 수행하실 수 있습니다.

기능 설명
Instance Monitor
(Database Monitor)		 인스턴스 운영 관리 통계정보 중 가장 중요하고 자주 보는 핵심적인 통계정보를 9개의 그래프를 이용하여 파악할 수 있습니다. 파일 저장 기능을 지원하여 과거의 통계정보를 나타내는 그래프를 언제든지 다시 확인 할 수 있습니다.
Wait Event Monitor 일시적인 성능저하 상황에서 Wait Event를 모니터링 할 수 있습니다. 초단위로 갱신 간격을 설정하여 주어진 시간 간격에 따라 변화하는 Wait Event 정보를 리스트, 그래프와 차트를 통해 한눈에 볼 수 있는 기능을 제공합니다.
ASH Monitor 최근 1시간내의 ASH(Active Session History) 정보를 이용하여 세션 레벨 실시간 모니터링을 지원합니다. 특정 구간 선택 시에 선택된 구간에서 수행되었던 Top SQL, Session, Wait Event 정보를 통해 현재뿐 아니라 과거시점에 발생한 장애 및 성능 저하 원인까지 세션 레벨로 분석할 수 있게 도와준다. 10g부터 사용 가능합니다.
Space Manager 테이블 스페이스와 데이터 파일에 대한 사용현황을 다양한 방식으로 제공합니다. 테이블 스페이스 별로 저장된 세그먼트에 대한 정보를 세그먼트 타입 별, 사용자 별로 제공하는 등 제반 상황을 일목요연하게 확인하게 체계적으로 관리할 수 있습니다.
Security Manager 데이터베이스에 정의된 사용자, 롤, 권한, 프로파일 등의 모든 보안 관련 사항을 다양한 관점에서 손쉽게 조회할 수 있으며, 새로운 사용자의 추가 및 권한 부여/회수를 비롯한 사용자 정보 변경을 손쉽게 할 수 있습니다.
Stats Manager 비용 기반 옵티마이저(Cost-Based Optimizer) 운영 환경에서 테이블, 인덱스, 클러스터와 같은 세그먼트 오브젝트에 대한 통계 정보를 다양한 옵션을 통해서 손쉽게 관리할 수 있습니다.
Health Check 성능 관점에서 데이터베이스 부하의 요소를 도출하고 분석하여 문제를 개선하기 위한 Online 점검 서비스로서 데이터베이스 운영에 대한 핵심 요소들의 건강 척도를 알 수 있습니다.
Graph & Report 데이터베이스 운영 현황 및 성능을 파악하기 위한 Pre-defined/User-defined SQL의 수행을 통하여 그 결과를 다양한 형태의 Graph와 Report 로 제공합니다
Log Miner 데이터베이스의 변경 내역을 저장하고 있는 Redo Log을 다양한 옵션으로 조회할 수 있고 Update, Insert, Delete 등의 변경 사항을 복구 할 수 있는 기능을 제공합니다.
AWR Manager 데이터베이스 관리자가 주로 사용하며 데이터베이스의 성능을 리포트 형식으로 모니터링 할 수 있습니다. 리포트 종류는 ADDM(Auto Database Diagnostic Monitor), AWR(Automatic Workload Repository), AWR SQL, AWR Diff, ASH(Active Session History) 리포트가 있습니다.
Data Quality
Manager		 Data 자체의 검증을 통해 Data 결함을 분석하고 이후 에러를 발생 시킬 수 있는 요소들을 분석차원에서 지원합니다. 분석항목 – Distinct Count / Null count / Space Count / Minimum Value / Maximum Value / 각종 Data 분석지원. 또한 이러한 요소 데이터를 기반으로 하여 Summary / Pattern / Distinct Value List 및 Chart 등의 분석 자료를 제공합니다. 사용자가 분석에 필요한 업무 규칙에 대해 SQL 을 입력하여, Business Rule 의 등록 및 실행 결과 자료를 제공합니다.
  • Orange의 Reorg Manager 란?
    Orange Reorg Manager 는 잦은 Insert, Update, Delete 작업으로 인하여 단편화가 많이 진행된 테이블을 Downtime을 소화하면서 Online 상태에서 테이블을 다시 생성함으로써 불필요한 디스크의 낭비를 방지하고 조회 성능을 향상시키는데 유용한 툴이다. 테이블과 함께 인덱스, 제약조건, 트리거, 주석, 오브젝트 권한, 통계정보를 모두 한번의 과정으로 옮겨주기 때문에 수 작업으로 스크립트를 생성하여 하는 것보다 훨씬 쉽고, 효율적이다. 또한, Reorg Manager는 Oracle이 9i 부터 제공하는 DBMS_REDEFINITION 패키지를 활용함으로써 안정적인 성능을 보장합니다.
  • Online 상태에서 진행되므로 Downtime 최소화
    Reorganization 작업이 온라인 상태에서 진행되기 때문에 Database 서버의 Downtime을 최소화 할 수 있으므로 24시간 운영되어야 하는 시스템에 적합하다.
