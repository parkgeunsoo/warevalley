Trusted Orange

데이터베이스 작업 결재 솔루션

데이터베이스 종합 관리 솔루션

  • 기업의 중요 데이터를 안전하게 처리하는 업무 프로세스 절차를 제공하는데이터베이스 작업결재 솔루션
  • 컴플라이언스를 준수하는 절차를 적용하여 보안을 강화하고, 사용자들의업무 소통을 통하여 협업이 가능하도록 지원
  • 데이터 베이스 대표 개발툴인 Orange로 업무 편의성을 유지하면서 안전한 작업이 가능하도록 업무 균형을 유지

작업 결재 및 사전 통제 솔루션

  • Trusted Orange 는 중요 데이터 작업 처리 시에 결재 프로세스를 제공
  • 사용자별 또는 사용자 그룹별로 데이터베이스 접근 권한을 부여
  • SQL 타입별 사용 권한 제어 정책을 사용하여 데이터의 변경 전/후 보안 기능을 제공
  • 모든 관리대상 데이터베이스의 작업은 결재권자의 결재 승인을 받은 후 실행 가능하므로, 데이터베이스 작업에 대한 사전 통제가 가능
  • 관리대상 데이터베이스에서 실행된 모든 작업에 대하여 감사로그를 기록함으로써 사후 보안 기능을 제공
  • 사전/사후 보안 기능은 기존 데이터베이스 환경에서 발생할 수 있는 보안상의 취약성과 결함 보완
